pySCIPAC

Python implementation of SCIPAC (Single Cell Identifier of Phenotype-Associated Cells) for identifying single cells associated with clinical phenotypes using bulk RNA-seq data.

Get Started View on GitHub

Features

Phenotype Association

Identify single cells associated with clinical phenotypes like disease subtypes, treatment response, or survival outcomes.

Multiple Phenotype Types

Supports binary (binomial), continuous (gaussian), survival (cox), and ordinal (cumulative) phenotypes.

Statistical Inference

Bootstrap-based confidence intervals and significance testing for robust statistical inference.

Scanpy Integration

Seamless integration with scanpy and AnnData for modern single-cell analysis workflows.

Installation

Install pySCIPAC from PyPI:

pip install pyscipac

Or install from GitHub for the latest development version:

pip install git+https://github.com/mianaz/pySCIPAC.git
